## Why Azure App Service Instead of a Virtual Machine

Azure App Service was selected because:

- Fully managed platform (no OS maintenance)
- Easy deployment and scaling
- Built-in monitoring and logging
- Cost-effective for small web applications
- Faster setup compared to provisioning a VM
